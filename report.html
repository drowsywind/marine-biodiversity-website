1-1. 背景と目的

沿岸域は、陸から流れ込む水や栄養分と海の潮流が出会う場所であり、魚類、貝類、海藻、 鳥類、微小生物などが利用する重要な生息環境である。一方で、海洋ごみ、踏み荒らし、 沿岸開発、海水温の変化などの影響を受けやすい。本調査では、生物、漂着ごみ、環境条件を 同じ範囲・同じ方法で記録し、海洋生物多様性を「生態系・種・遺伝子」の三つの観点から考察する。

3. 考察

3-1. 三つの生物多様性から見た結果

多様性のレベル本調査で確認・検討した内容保全上の意味
生態系の多様性 波打ち際、漂着物帯、乾いた砂地、海浜植生という異なる環境を区別した。 海岸を一様な場所として扱わず、それぞれの環境を残す必要がある。
種の多様性 植物、甲殻類、軟体動物、鳥類、海藻、魚類の存在または痕跡を記録した。 目立つ大型生物だけでなく、小型生物や生息痕跡も評価対象にする。
遺伝子の多様性 現地の外見観察だけでは直接評価できないため、貝殻模様などの個体差を補助記録とした。 遺伝子多様性は見た目だけで断定せず、専門的な調査が必要である。

3-2. 海岸ごみと生物への影響

モデル集計では、人工物60個のうちプラスチック片・包装が31個で最も多かった。 小さな破片は回収が難しく、生物が餌と間違える可能性もある。漁糸やロープは、 鳥類、魚類、ウミガメなどが絡まる原因になり得る。また、ごみは海岸で捨てられたものだけでなく、 街や河川から流入し、潮流や風によって移動した可能性がある。

3-3. 調査の限界

  1. 一回の調査であり、季節・潮位・天候による変化を比較できていない。
  2. 目視調査のため、砂中、夜間、顕微鏡レベルの生物は評価できていない。
  3. 種の同定には不確実性があり、写真と専門家による確認が必要である。
  4. 本ページの個体数と環境値はモデルデータであり、現地の実態を示すものではない。

4. 保全提案と今後の計画

4-1. 地域で実施できる行動

  1. 定点観察：同じ50 m区画を毎月または季節ごとに調査し、写真位置も固定する。
  2. ごみの発生源対策：海岸清掃だけでなく、河川、道路、観光利用時のごみ削減を進める。
  3. 生息場所への配慮：海浜植物やカニ類の巣穴がある区域を踏み荒らさない。
  4. 情報共有：調査結果をウェブサイト、展示パネル、パンフレットで公開する。
  5. 外部連携：行政、市民団体、漁業者、研究機関と連携して同定と調査方法を改善する。
